example code:


          <!DOCTYPE html>
          <html lang="en" dir="ltr">

              <meta charset="utf-8">

            <body>

            </body>
          </html>
<!DOCTYPE html>
- tells the interpreter what type of document you are creating
<html></html>
- this is used to determine the beginning and end of your html document.
<head></head>
- this tag determines the head of your document where you can add your character
set, the title of your webpage and links to your css external files as well as js files.
<body></body>
- this tag indicates the content of your webpage which will be displayed.

Base Structure Code
Tag Purpose
<p> this tag is for a paragraph if you want to start a new line the the <br> tag
<pre> this is for pre formatted text. It will print exactly as inputed including line breaks
<h1> to <h6> These are for different sizes of headers h1 being largest font.
<div> or <section> both are used to indicate blocks of text except section has a smaller font size.
<a> This tag is used to create hyper links
<img> this is used to create an image, size can be modified with css
<video> used to input an imbedded video into your webpage
<iframe> used to display a webpage in a webpage but is really useful for youtube videos.
<i> <em> these apply italics to text <em> is for emphasizing importance
<b> <strong> both apply bold to text <strong> is for strong importance
<header> <footer> to create a header/footer for your page
<q> <blockquote> blockquote for a large quote from external source q applies quotation marks
<span> to apply styling over certain text
<mark> usually used for highlighting certain text
